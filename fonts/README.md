# Fonts Directory

Place your `.ttf` or `.otf` font files here, organized by font family name.

## Expected Structure

```
fonts/
├── Amiri/
│   └── Amiri-Regular.ttf
├── Cairo/
│   └── Cairo-Regular.ttf
└── README.md
```

## Where to Get Fonts

- [Google Fonts](https://fonts.google.com/) — search for "Amiri", "Cairo", "Tajawal", etc.
- Download the font family ZIP, extract it, and place the `.ttf` files in a subfolder named after the font.
